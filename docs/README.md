## 快速开始

### 1. 环境准备

```bash
# 安装 Go 1.24.2+
go version

# 克隆项目
git clone https://github.com/charmbracelet/mods.git
cd mods
```

### 2. 安装依赖

```bash
go mod download
go mod tidy
```

### 3. 构建运行

```bash
# 构建
go build -o mods .

# 运行
./mods "你好，世界"
```

### 4. 配置 API 密钥

```bash
# 设置环境变量
export OPENAI_API_KEY="sk-..."

# 或编辑配置文件
./mods --settings
```

## 核心概念

### 架构层次

```
┌─────────────────────────────────────────────────────────┐
│                    用户界面层                            │
│  CLI 入口 │ 交互式表单 │ 终端渲染器                      │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                    应用核心层                            │
│  配置管理 │ 消息处理 │ 状态管理                          │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                    服务集成层                            │
│  OpenAI │ Anthropic │ Cohere │ Google │ Ollama │ MCP   │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                    数据存储层                            │
│  SQLite DB │ 文件缓存 │ 配置文件                         │
└─────────────────────────────────────────────────────────┘
```

### 支持的 AI 服务

| 服务商 | 模型示例 |
|--------|----------|
| OpenAI | GPT-4o, GPT-4, GPT-3.5, o1, o3 |
| Anthropic | Claude Sonnet 4, Claude Opus 4 |
| Google | Gemini 1.5 Pro, Gemini 2.0 Flash |
| Cohere | Command-R, Command-R-Plus |
| Ollama | Llama 3.2, 本地模型 |

## 贡献指南

### 如何贡献

1. Fork 项目仓库
2. 创建功能分支 (`git checkout -b feature/amazing-feature`)
3. 提交更改 (`git commit -m 'feat: 添加某功能'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 创建 Pull Request

### 代码规范

- 遵循 [Effective Go](https://golang.org/doc/effective_go) 指南
- 使用 `gofmt` 格式化代码
- 通过 `golangci-lint` 检查
- 添加必要的测试和文档

详细规范请参考 [开发规范](./development/01-开发规范.md)。
